# 🔥 BlazeVision – Fire Detection System

BlazeVision is a computer vision web application that detects fire in images using a trained YOLO deep learning model.  
The system allows users to upload an image and automatically detects fire using AI and highlights it in the output image.

This project combines **Flask, OpenCV, and YOLO (Ultralytics)** to build an intelligent fire detection system.

---

## 🚀 Features

- Fire detection using a trained YOLO model
- Upload images through a web interface
- Real-time image processing
- Annotated detection results
- Simple and clean web interface
- Supports JPG, JPEG, PNG images

---

## 🧠 Technologies Used

- Python
- Flask
- OpenCV
- NumPy
- Ultralytics YOLO
- HTML/CSS
- Torch / TorchVision
